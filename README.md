# MATLAB Code for Green Supply Chain Finance Simulation

This repository contains the MATLAB code used in the simulation analysis for the manuscript:

**Avoiding Greenwashing Behaviors in Green Supply Chain Finance: The Role of Blockchain Technology**

## Files
- `Simulations.m`: main simulation script
- `green.m`: replicator dynamics before blockchain adoption
- `green2.m`: replicator dynamics after blockchain adoption

## Notes
- The code is used to generate the numerical simulation results reported in Section 5 of the manuscript.
- Parameter settings are described in the paper.
- MATLAB version: 2023a
